= Quick start

== Requirements

* Docker
* Docker Compose
* Rust 1.78+
* Cargo

== Dependencies

[bash]
----
rustup update stable
----

== Setup project

* clone repository
* `devops/scripts/setup.sh`

== Start Databases

[bash]
----
docker compose up
----

== Run demo web application:

[bash]
----
cd demo-webapp
cargo run
----

Now you can see API documentation and send requests via Swagger-UI. link:http://localhost:8083/[]

== Or run demo cli application:

[bash]
----
cd demo-app
cargo run
----
